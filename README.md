# ArduinoStopwatch
Basic counting stopwatch for Arduino

So you want to Build an Arduino Stopwatch! That's awesome and a great starting project for anyone!  

---

## Table of Contents
1. Necessary Items  
2. Getting Started  


### Necessary Items
There are a few things you are going to need to get started!  
1. Arduino UNO Board [Amazon](https://www.amazon.com/gp/product/B008GRTSV6/ref=ppx_yo_dt_b_asin_title_o04_s00?ie=UTF8&psc=1)  
    - Purchased on Amazon for around $20 on Sale and $30 MSRP  
2. DFROBOT Gravity: 1602 LCD Keypad Shield [Amazon](https://www.amazon.com/gp/product/B006D903KE/ref=ppx_yo_dt_b_asin_title_o04_s00?ie=UTF8&psc=1)  
    - Used to display stopwatch letters on screen  
3. Standard USB-DataSync Cable for Arduino UNO [Amazon](https://www.amazon.com/gp/product/B08RCJXY1Z/ref=ppx_yo_dt_b_asin_title_o05_s00?ie=UTF8&psc=1)  

You will also need a computer where you can connect your Arduino and get started!
